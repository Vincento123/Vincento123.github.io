05 — Projects

Projects & Process Optimisation

CASE STUDY 01 · ACCOUNTS PAYABLE
Accounts Payable Workflow Improvement
Problem
At Digital Qatalyst, the accounts payable process relied heavily on manual invoice tracking across multiple email threads and spreadsheets — creating reconciliation delays, missed cut-offs, and data inconsistencies that complicated financial reporting.
Action
Migrated the full AP lifecycle into Microsoft Dynamics 365 Business Central — centralising invoice processing, automating approval workflows, enforcing cut-off timelines, and building a structured payables tracker with vendor reconciliation templates in Excel for cross-validation. Implemented a standardised documentation protocol aligned to IFRS and GAAP requirements to support audit readiness.
Result
Reduced manual AP processing time by approximately 30%, eliminated recurring reconciliation discrepancies, and strengthened the organisation’s financial data integrity. Month-end close became more predictable and audit-ready documentation improved significantly.
Accounts Payable ERP Systems Automation Financial Data IFRS Compliance
CASE STUDY 02 · PAYROLL
Payroll Process Accuracy Improvement
Problem
Payroll processing involved fragmented data sources — employee records, PAYE schedules, NSSF rates, SHIF contributions, and the Affordable Housing Levy — managed across separate spreadsheets without a unified workflow, leading to calculation errors, late statutory remittances, and compliance risks.
Action
Designed and implemented a structured payroll automation model in Excel integrating all statutory deductions (PAYE, NSSF, SHIF, AHL) into a single master payroll workbook with formula-driven calculations, automated variance flags, and built-in reconciliation checks. Introduced a payroll run checklist aligned to IFRS 19 and IAS 12 for audit compliance. Synced payroll data with the ERP system for real-time GL posting.
Result
Achieved zero payroll calculation errors across subsequent payroll cycles. Statutory remittances became consistently on-time, eliminating penalty risk. Processing time per cycle reduced significantly, and payroll reporting became a clean, reliable input into monthly financial analysis and forecasting.
Payroll Automation Tax Compliance Financial Reporting Forecasting Data Management
CASE STUDY 03 · SAAS & COST GOVERNANCE
SaaS Cost Optimisation & Financial Data Governance
Problem
The organisation’s SaaS subscription portfolio had grown organically without systematic oversight — redundant tools, underutilised licences, and overlapping renewals were creating avoidable expenditure and gaps in financial data security controls around system access and user provisioning.
Action
Conducted a full SaaS audit — cataloguing all active subscriptions, licence counts, renewal dates, and cost per user. Built a governance dashboard in Excel tracking utilisation metrics and renewal alerts. Performed cost-benefit analysis on each tool against defined business requirements. Simultaneously introduced financial data security controls — access reviews, user deprovisioning checklists, and data handling protocols aligned to cybersecurity awareness best practices.
Result
Identified and eliminated redundant SaaS tools, reducing the subscription footprint and delivering measurable cost savings on recurring expenditure. Established an ongoing renewal governance process preventing future unplanned renewals. Improved financial data security posture through structured access controls and documented deprovisioning workflows.
